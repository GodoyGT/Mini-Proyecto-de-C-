# MINI PROGRAMA EN C++ - Sistema de Control de Empleados

## Descripción
Este proyecto es un sistema básico de control de empleados desarrollado en C++.  
El programa funciona en consola y permite registrar información de empleados, almacenarla y mostrar diferentes resultados al usuario.

Este proyecto fue realizado utilizando temas básicos de programación como:
- variables
- arreglos
- funciones
- ciclos
- condicionales
- validaciones

---

## Funcionalidades principales

El sistema permite:

- Registrar empleados
- Mostrar empleados registrados
- Buscar empleados por nombre
- Calcular el sueldo total
- Mostrar el empleado con mayor sueldo
- Mostrar el empleado con menor sueldo
- Validar datos incorrectos

---

## Estructura básica del proyecto

El programa está organizado mediante funciones:

- `registrarEmpleados()`
- `mostrarEmpleados()`
- `buscarEmpleado()`
- `calcularSueldos()`
- `mayorSueldo()`
- `menorSueldo()`

Además, se utilizan arreglos para almacenar:
- nombres
- edades
- sueldos
